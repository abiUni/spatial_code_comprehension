Code Navigation and Annotation Tasks

Overview:

Next, you will be looking at the source code of the Flutter Folio App.

The Flutter project will be opened in the code documentation tool introduced previously.

You will be asked to complete three tasks, in which you will navigate through the code base and annotate certain code lines.

Each task comprises two parts:

  • The first part is a navigation part where you are asked to trace and annotate a given feature throughout the code (example from user interaction to data change).

  • The second part is a comprehension part where you are either asked to find and annotate the code line(s) responsible for a certain bug affecting the feature you traced or to annotate the lines where a certain logic is implemented.
Please answer as accurately as possible and try to annotate as many relevant code sections as possible.
If you get stuck on a task, you can skip to the next task by clicking "Continue". You cannot go back to the previous task.

We encourage you to attempt all tasks.

Please use the participant ID assigned to you (see invitation email) to login to the tool on the next page.

Good Luck!

Reminder of Tool Controls:

Use trackpad or scrollwheel (press scrollwheel to drag) for movements on the canvas.

Zoom-in/zoom-out works using scrollwheel+alt or scrollwheel + option ⌥ on mac.

Using the trackpad keep alt or option ⌥ pressed while moving your fingers over the trackpad to zoom in and out of the canvas.
