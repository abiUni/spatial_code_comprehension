Online Study on Visualisation for Code Understanding

Dear Participant,

Thank you for your interest in our experiment!
We are investigating code visualisations to help developers with code comprehension and navigation.
We are very grateful that you're helping us by participating in this experiment.

Experiment Overview:

In this experiment, we will ask you to solve three tasks in which you will be searching through a given code base, the Flutter Folio App, and annotating relevant code lines. A web application for code documentation will be used to visualise the code. We will then ask you to take a cognitive memory task and finally to compelete a survey including demographics and an evaluation of the code documentation tool.

Please note that for legal reasons you are only allowed to partcipate in the experiment if you are 18 years or older.

With this experiment:

  • we compare the outcome of the three code searching tasks under different circumstances
  • we log the interactions with the tool (annotation events, file interactions, etc) during the tasks to better understand which features are helpful
  • your data will only be stored and used in anonymized form
  • you will get the chance to win one Voucher

The code you will be searching through (Flutter Folio App) is a project using the Flutter framework, so basic Dart skills are sufficient.

Please stay focused on the experiment until it ends, otherwise we won't be able to use your data.

Please make sure you have at least one hour free to do the experiment.

Please don't interrupt the experiment for longer periods of time once you've started.

You are free to leave the experiment at any time, your participation will then not be counted.

For optimum experience, we recommend keeping your browser window as large as possible.

Data Protection Notice:

The data gathered in this experiment will be used solely for research purposes and will not be made available to any third party. The participation in this study is voluntary and the participants can leave the experiment at any time. The data of each participant will be anonymized using an identification number. Each participant is invited to provide their email adress if they wish to take part in the voucher lottery, no trace link will be made between email adresses and identification numbers of the experiment. The scientific results of the experiment will be published under full anonymity for the participants.
